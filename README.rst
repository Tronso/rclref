rclref
===========

A riak_core application

Build
-----

::

    make release

Test
----

::

    make ct

Run
---

::

    make console

Try
---

::

    1> rclref:ping().
    {pong,'rclref1@127.0.0.1', 9...8}

Quit
----

::

    2> q().
